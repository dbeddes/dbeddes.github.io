Hi my name is Shawn Beddes and I've been working with SQL Server now for almost 20 years. Like many of you I have worked often as both a production DBA and a database developer.

Having to don the hat of multiple roles I have had to learn many things about how SQL Server works and how to overcome various different issues over time.  My goal with this blog is to help everyone I can with information on how I've overcome issues that I've faced, and to also just show off of the neat things that can be done with SQL Server.
